2013-07-2
========================================================================

设M,X,R,F,V六个功能键
功能键M: 功能开关，按下键按　电阻(R) ->电容(C) ->电感(L) ->自动(AUTO) 顺次轮换切换，长按功能键则进入系统设置(S)模式。
模式键X: 串并联切换开关，按下键按　串联(S) -> 并联(P) 顺次换切。
量程键R: 量程开关，按下键按　40 -> 1K ->10K ->100K 　顺次轮换切换。
频率键F: 频率开关，按下键按 100Hz -> 1000Hz -> 7800Hz　顺次轮换切换。
清零键C: 清零，当表笔开路时，开路清零，当表笔短路时，短路清零，　注意短按只清当前量程的，长按（１秒以上）则清所有量程。
数值键V: 记忆比较功能，按下记录当前测量值并进入记忆比较模式，在这一模式下，之后的测量值都会和记忆值比较交显示偏差绝对值及百分比，再按则退出记忆比较模式。

========================================================================


b11

-诊断更新时间现在等于从菜单上的AVERAGES选项。


b10


-在诊断中，现在读数平均为16个值，被认为是标准的


在诊断屏幕上的平均和超临界值，中间模式）（MEAN）和平均方差（SIGMA）切换到REL

在屏幕上的REL图标意味着2和7通道的值不是绝对值，而是从第一频道的偏差，包括在SIGMA模式中，在该模式中，企业社会责任是用来区分频道而不是自己的频道。这是通过按下复选框切换模式。


zy2是所有三个通道的振幅的一次跳跃，它可以看到，这是一个异常的增长，只要你不注意它，等待它的疲劳到一个小的值。

Z2 7通道，在那里可以进行反转，以理解与第一频道的相位差，所以0度相实际上是-180g。

zy3第一频道绝对阶段的起始值）和其他频道的起始值可能会在开机后发生变化，只有两个不同的值，这是正常的。

-增加了第三个参数显示

-在诊断屏幕上你可以用SP按钮切换频率

-相位诊断现在相对零通道

-删除7频道的葡萄糖。


在寻找这件事的来源，发现了一个相当不愉快的葡萄干，导致了错误的测量7通道，并导致负电阻97k线圈。现在，在这种情况下，振幅均匀地被跳跃切换到100微米左右。尝试一个新的固件或不。
